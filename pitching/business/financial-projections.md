# Financial Projections

## Revenue Model Summary

CrowdFUNding generates revenue through a "Low Tax, High Volume" model on three streams.

| Revenue Stream          | Margin        | Description                              |
| ----------------------- | ------------- | ---------------------------------------- |
| **Platform Fee (2.5%)** | Volume-driven | Per-donation fee (50% below competitors) |
| **Digital Marketplace** | 95%+          | NFT cosmetics for Impact Pets            |
| **B2B API (Future)**    | SaaS          | CSR dashboards for corporates            |

| -------------------- | ------------------ |

| **Net per Donation** | Rp 1,725           |

---

## Why Base Makes This Possible

Traditional payment processors charge **5-15%** fees, making micro-donations unprofitable.  
Base L2's near-zero gas (~$0.01) enables:

- **Viable micro-donations** (Rp 10,000 / ~$0.65)
- **50% lower platform fees** than competitors
- **Profitable unit economics** at high volume
