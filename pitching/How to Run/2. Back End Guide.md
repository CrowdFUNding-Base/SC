# 2. Back End Guide

The backend handles fiat payments encryption, metadata storage, and off-chain gamification logic.

## Prerequisites

- Node.js (v18+)
- npm or yarn

## Setup

1.  **Navigate to Backend**

    ```bash
    cd backend
    ```

2.  **Install Dependencies**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Configure Environment**
    Create a `.env.local` file:
    ```
    MIDTRANS_SERVER_KEY=...
    MIDTRANS_CLIENT_KEY=...
    DATABASE_URL=...
    RPC_URL=...
    PRIVATE_KEY=... (For minting on behalf of fiat users)
    ```

## Running Locally

Start the development server:

```bash
npm run dev
```

The server will start at `http://localhost:3000/api`.

## Key Endpoints

- `POST /api/payment/token`: Generate Midtrans token.
- `POST /api/payment/notification`: Webhook for Midtrans.
- `GET /api/campaigns`: Fetch campaign lists.
